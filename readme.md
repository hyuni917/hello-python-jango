
###### 가상환경 설정을 위한 패키지 설치
```
> pip3 install virtualenv
```

###### 가상환경 생성
```
> virtualenv venv
```

###### 가상환경 활성화
```
> (mac) source venv/bin/activate
> (win) venv/bin/activate
```

###### 장고 설치
```
> pip3 install django
```

###### 장고 프로젝트 생성
```
> django-admin startproject community
> cd community
```
